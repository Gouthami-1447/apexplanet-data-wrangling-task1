# 📊 Data Immersion & Wrangling – Task 1

This repository contains Task 1 of my **Data Analytics Internship at ApexPlanet Software Pvt. Ltd.**.  
The focus of this task is on understanding raw data and preparing it for analysis through data cleaning and preprocessing.

---

## 🚀 Project Overview

In this project, I worked on:
- Exploring and understanding the dataset
- Creating a data dictionary for better clarity
- Identifying data quality issues (missing values, duplicates, inconsistencies)
- Cleaning and transforming the data using Python (Pandas)
- Preparing a final dataset ready for analysis

---

## 🛠️ Tools & Technologies Used

- Python 🐍
- Pandas 📊
- Jupyter Notebook / Google Colab

## 📂 Key Steps

1. Data Collection & Loading  
2. Data Exploration  
3. Data Cleaning  
4. Data Transformation  
5. Final Dataset Preparation
